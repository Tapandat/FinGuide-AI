
# FinGuide AI 💰

FinGuide AI is a smart financial expense tracking and analytics system built using Python and Streamlit.

This project helps users:
- Track expenses
- Manage income
- Analyze spending habits
- Predict future expenses using Machine Learning
- Receive financial reports through Email

---

# Features

✅ User Login & Registration  
✅ Income Management  
✅ Expense Tracking  
✅ Financial Analytics Dashboard  
✅ Budget Planning  
✅ AI Expense Prediction  
✅ Email Reports using SMTP  
✅ PDF Financial Reports  
✅ Multi-user Support  
✅ Professional FinTech UI

# How to Run the Project

## Step 1 — Install Libraries

```bash
pip install -r requirements.txt
```

---

## Step 2 — Run the Application

```bash
streamlit run app.py
```
